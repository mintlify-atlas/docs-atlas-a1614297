## Overview

tmux uses a prefix key system for key bindings. By default, the prefix is `C-b` (Ctrl+b), followed by a command key. Key bindings are stored in key tables and can be customized to match your workflow.

## Key Binding System

### Key Tables

tmux organizes key bindings into named tables. The main tables are:

- **root** - Default key table (no prefix required)
- **prefix** - Keys that follow the prefix key
- **copy-mode** - Emacs-style copy mode bindings
- **copy-mode-vi** - Vi-style copy mode bindings

<Note>
Key tables are implemented as red-black trees in `key-bindings.c` for efficient lookup and organization.
</Note>

### Binding Keys

Use the `bind-key` command (alias: `bind`) to create custom bindings:

<CodeGroup>

```bash Basic Binding
# Bind r to reload configuration
bind r source-file ~/.tmux.conf
```

```bash With Description
# Bind with description (-N flag)
bind -N 'Reload configuration' r source-file ~/.tmux.conf
```

```bash Different Table
# Bind in root table (no prefix needed)
bind -n C-Left select-pane -L
```

```bash Repeatable Binding
# Allow repeating without prefix (-r flag)
bind -r C-Up resize-pane -U 5
```

</CodeGroup>

### Pane Management

<Steps>

<Step title="Split Panes">
```bash
# Split horizontally (left/right)
bind % split-window -h

# Split vertically (top/bottom)
bind '"' split-window
```
</Step>

<Step title="Navigate Panes">
```bash
# Arrow keys (with repeat)
bind -r Up select-pane -U
bind -r Down select-pane -D
bind -r Left select-pane -L
bind -r Right select-pane -R

# Next pane
bind o select-pane -t:.+

# Last pane
bind \; last-pane
```
</Step>

<Step title="Resize Panes">
```bash
# Resize by 5 cells
bind -r M-Up resize-pane -U 5
bind -r M-Down resize-pane -D 5
bind -r M-Left resize-pane -L 5
bind -r M-Right resize-pane -R 5

# Resize by 1 cell
bind -r C-Up resize-pane -U
bind -r C-Down resize-pane -D
bind -r C-Left resize-pane -L
bind -r C-Right resize-pane -R
```
</Step>

<Step title="Zoom & Swap">
```bash
# Zoom pane
bind z resize-pane -Z

# Swap panes
bind '{' swap-pane -U
bind '}' swap-pane -D
```
</Step>

</Steps>

### Copy Mode Keys

From `key-bindings.c:350-655`, the default copy mode bindings:

<CodeGroup>

```bash Enter Copy Mode
# Enter copy mode
bind [ copy-mode

# Enter copy mode and scroll up
bind PPage copy-mode -u
```

```bash Vi-style Navigation
# In copy-mode-vi table
bind -Tcopy-mode-vi v send -X begin-selection
bind -Tcopy-mode-vi y send -X copy-selection-and-cancel
bind -Tcopy-mode-vi C-v send -X rectangle-toggle

# Movement
bind -Tcopy-mode-vi h send -X cursor-left
bind -Tcopy-mode-vi j send -X cursor-down
bind -Tcopy-mode-vi k send -X cursor-up
bind -Tcopy-mode-vi l send -X cursor-right
```

```bash Emacs-style Navigation
# In copy-mode table
bind -Tcopy-mode C-Space send -X begin-selection
bind -Tcopy-mode M-w send -X copy-selection-and-cancel
bind -Tcopy-mode C-g send -X clear-selection

# Movement
bind -Tcopy-mode C-f send -X cursor-right
bind -Tcopy-mode C-b send -X cursor-left
bind -Tcopy-mode C-n send -X cursor-down
bind -Tcopy-mode C-p send -X cursor-up
```

</CodeGroup>

## Customization Examples

### Custom Prefix Key

From `example_tmux.conf:28-31`:

```bash
# Change prefix from C-b to C-a
set -g prefix C-a
unbind C-b
bind C-a send-prefix
```

<Tip>
Changing the prefix to `C-a` is popular among GNU Screen users for familiarity.
</Tip>

```bash
# Toggle monitoring activity
bind m set monitor-activity

# Toggle synchronized panes with feedback
bind y set synchronize-panes\; display 'synchronize-panes #{?synchronize-panes,on,off}'
```

### Mouse Integration

From `example_tmux.conf:33-36`:

```bash

```

## Advanced Key Binding Techniques

### Key Sequences

Create multi-key sequences using the `-T` flag:

```bash
# Create a custom key table
bind -Ttable2 c list-keys
bind -Ttable1 b switch-client -Ttable2
bind -Troot a switch-client -Ttable1

# Now typing 'a b c' will run list-keys
```

### Conditional Bindings

```bash
# Different bindings based on pane mode
bind -n WheelUpPane if -F '#{||:#{alternate_on},#{pane_in_mode},#{mouse_any_flag}}' {
    send -M
} {
    copy-mode -e
}
```

### Repeat Bindings

From `key-bindings.c:190-224`, the `-r` flag enables repeat:

```bash
# Allow repeating without pressing prefix again
bind -r C-Up resize-pane -U
bind -r C-Down resize-pane -D

# After pressing prefix once, you can press C-Up multiple times
```

<Note>
Repeat timeout is controlled by the `repeat-time` option (default: 500ms).
</Note>

## Managing Key Bindings

### List All Bindings

```bash
# List all bindings
tmux list-keys

# List bindings for specific table
tmux list-keys -T copy-mode-vi

# List with descriptions
tmux list-keys -N
```

### Unbind Keys

```bash
# Unbind a key
unbind C-z

# Unbind from specific table
unbind -T copy-mode-vi Space

# Unbind all keys in table
unbind -aT mytable
```

### Reset to Defaults

From `key-bindings.c:260-289`:

```bash
# Reset a single key to default
tmux bind-key -r C-Up

# This restores the default binding if it exists
```

## Key Binding Flags

<Accordion title="Available Flags">

| Flag | Description |
|------|-------------|
| `-r` | Key may repeat (within repeat-time) |
| `-n` | Bind in root table (no prefix needed) |
| `-N` | Add description for list-keys -N |
| `-T` | Specify key table |
| `-s` | Execute in the background |

</Accordion>

## Practical Examples

### Quick Split Navigation

```bash
# Smart pane switching with awareness of Vim splits
bind -n C-h run "(tmux display-message -p '#{pane_current_command}' | grep -iq vim && tmux send-keys C-h) || tmux select-pane -L"
bind -n C-j run "(tmux display-message -p '#{pane_current_command}' | grep -iq vim && tmux send-keys C-j) || tmux select-pane -D"
bind -n C-k run "(tmux display-message -p '#{pane_current_command}' | grep -iq vim && tmux send-keys C-k) || tmux select-pane -U"
bind -n C-l run "(tmux display-message -p '#{pane_current_command}' | grep -iq vim && tmux send-keys C-l) || tmux select-pane -R"
```

### Layout Presets

From `key-bindings.c:414-420`:

```bash
# Preset layouts
bind M-1 select-layout even-horizontal
bind M-2 select-layout even-vertical
bind M-3 select-layout main-horizontal
bind M-4 select-layout main-vertical
bind M-5 select-layout tiled
bind M-6 select-layout main-horizontal-mirrored
bind M-7 select-layout main-vertical-mirrored
```

### Copy to System Clipboard

```bash
# Copy to system clipboard (Linux)
bind -T copy-mode-vi y send -X copy-pipe-and-cancel "xclip -selection clipboard"

# Copy to system clipboard (macOS)
bind -T copy-mode-vi y send -X copy-pipe-and-cancel "pbcopy"

# Copy to system clipboard (Windows/WSL)
bind -T copy-mode-vi y send -X copy-pipe-and-cancel "clip.exe"
```

<Tip>
Use the `-P` flag with copy commands to prevent setting the paste buffer, useful when only piping to external commands.
</Tip>

## Troubleshooting

### Key Not Working

<Steps>

<Step title="Check if key is bound">
```bash
tmux list-keys | grep "your-key"
```
</Step>

<Step title="Check key table">
```bash
# Verify which table is active
tmux display-message -p "#{client_key_table}"
```
</Step>

<Step title="Test key code">
```bash
# See what tmux receives
tmux list-keys -N | grep "your-description"
```
</Step>

</Steps>

### Key Conflicts

```bash
# Find conflicting bindings
tmux list-keys -T root | grep "C-a"

# Unbind conflicting key
tmux unbind -T root C-a
```
