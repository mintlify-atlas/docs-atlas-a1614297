## Overview

This page covers fundamental tmux configuration options to customize your environment. These examples are practical, copy-pasteable configurations for common needs.

## Configuration File Location

tmux looks for configuration in these locations (in order):
- `~/.tmux.conf` (traditional location)
- `$XDG_CONFIG_HOME/tmux/tmux.conf` (modern XDG location)
- `/etc/tmux.conf` (system-wide configuration)

Create your configuration file:

```bash
touch ~/.tmux.conf
```

## Essential Configuration

### Change Prefix Key

The default prefix `C-b` (Ctrl-b) can conflict with other applications. Many users prefer `C-a`:

```conf
# Change prefix from C-b to C-a
set -g prefix C-a
unbind C-b
bind C-a send-prefix
```

<Tip>
The `send-prefix` binding allows you to send the prefix key to applications running inside tmux by pressing it twice (e.g., `C-a C-a`).
</Tip>

### Enable Mouse Support

Mouse support makes tmux more accessible and allows clicking to select panes and windows:

```conf
# Enable mouse support
set -g mouse on

# Disable copy mode dragging (optional)
unbind -n MouseDrag1Pane
unbind -Tcopy-mode MouseDrag1Pane
```

### Color and Terminal Settings

<CodeGroup>

```conf Basic Colors
# Enable RGB color support for xterm
set-option -sa terminal-features ",xterm*:RGB"

# Set default terminal to support 256 colors
set -g default-terminal "tmux-256color"
```

```conf Status Line Colors
# Set status line colors
set -g status-style bg=black,fg=white

# Highlight current window
set -g window-status-current-style bg=red,fg=white,bold
```

</CodeGroup>

### Bell and Notification Settings

Control how tmux handles system bells from applications:

```conf
# Disable all bell notifications
set -g bell-action none

# Alternative: Show bell in status line only
set -g bell-action other

# Alternative: Show bell only in current window
set -g bell-action current
```

### Window and Pane Behavior

```conf
# Keep windows around after the process exits
set -g remain-on-exit on

# Start window numbering at 1 instead of 0
set -g base-index 1

# Start pane numbering at 1 instead of 0
set -g pane-base-index 1

# Renumber windows when one is closed
set -g renumber-windows on
```

## Status Line Customization

### Basic Status Line

```conf
# Show time in status right
set -g status-right "%H:%M"

# Show session name in status left
set -g status-left "[#S] "

# Increase status left length
set -g status-left-length 20
```

### Conditional Status Line

Use conditional formatting to change appearance based on environment:

```conf
# Change status background to red if running inside tmux
%if #{TMUX}
set -g status-bg red
%endif

# Change status based on hostname
%if "#{==:#{host},myhost}"
set -g status-style bg=red
%elif "#{==:#{host},myotherhost}"
set -g status-style bg=green
%else
set -g status-style bg=blue
%endif
```

## Key Bindings

### Window Selection Keys

Bind function keys to select windows by number:

```conf
# Bind F1-F12 to select windows 10-21
bind F1 selectw -t:10
bind F2 selectw -t:11
bind F3 selectw -t:12
bind F4 selectw -t:13
bind F5 selectw -t:14
bind F6 selectw -t:15
bind F7 selectw -t:16
bind F8 selectw -t:17
bind F9 selectw -t:18
bind F10 selectw -t:19
bind F11 selectw -t:20
bind F12 selectw -t:21
```

### Pane Management Keys

```conf
# Split panes using | and -
bind | split-window -h
bind - split-window -v
unbind '"'
unbind %

# Switch panes using Alt-arrow without prefix
bind -n M-Left select-pane -L
bind -n M-Right select-pane -R
bind -n M-Up select-pane -U
bind -n M-Down select-pane -D
```

### Useful Utility Bindings

```conf
# Toggle synchronize-panes (send input to all panes)
bind y set synchronize-panes\; display 'synchronize-panes #{?synchronize-panes,on,off}'

# Toggle monitoring activity in a window
bind m set monitor-activity

# Reload configuration file
bind r source-file ~/.tmux.conf \; display "Config reloaded!"

# Toggle window size between smallest and largest
bind F set -w window-size
```

## Complete Basic Configuration Example

<Accordion title="View Complete Configuration">

```conf
# ~/.tmux.conf - Basic tmux configuration

# ========================================
# Prefix Key
# ========================================
set -g prefix C-a
unbind C-b
bind C-a send-prefix

# ========================================
# Mouse Support
# ========================================
set -g mouse on
unbind -n MouseDrag1Pane
unbind -Tcopy-mode MouseDrag1Pane

# ========================================
# Colors and Terminal
# ========================================
set-option -sa terminal-features ",xterm*:RGB"
set -g default-terminal "tmux-256color"

# ========================================
# Status Line
# ========================================
set -g status-right "%H:%M"
set -g window-status-current-style "underscore"

# Change status to red if nested tmux
%if #{TMUX}
set -g status-bg red
%endif

# ========================================
# Behavior
# ========================================
set -g bell-action none
set -g base-index 1
set -g pane-base-index 1
set -g renumber-windows on

# ========================================
# Key Bindings
# ========================================

# Split panes
bind | split-window -h
bind - split-window -v

# Function keys for window selection
bind F1 selectw -t:10
bind F2 selectw -t:11
bind F3 selectw -t:12

# Utility bindings
bind m set monitor-activity
bind y set synchronize-panes\; display 'synchronize-panes #{?synchronize-panes,on,off}'
bind r source-file ~/.tmux.conf \; display "Config reloaded!"
```

</Accordion>

## Testing Your Configuration

After editing your configuration file, you can:

<Tabs>
<Tab title="Reload Config">
Within tmux, press `prefix` + `:` then type:

```conf
source-file ~/.tmux.conf
```

Or use the reload binding if you configured it:

```bash
prefix + r
```
</Tab>

<Tab title="Test Before Starting">
Check your configuration for syntax errors:

```bash
tmux -f ~/.tmux.conf source-file ~/.tmux.conf
```
</Tab>

<Tab title="Start Fresh Session">
Start a new tmux session to test changes:

```bash
tmux new-session -s test
```
</Tab>
</Tabs>
